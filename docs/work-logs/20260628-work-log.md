# 20260628 作業ログ

## 概要

外部研究者（生物統計家）から提供された先進医療「実績報告」様式（様式第1号別添1）を題材に、`draft-sdtm-template` スキルで SDTM テンプレートのドラフト生成を1件分実施。RWD・観察モードで10テンプレートを作成し、JASPEHR template（aCRF）・設計サマリー・レビュー依頼書を整備、レビュー用 PR #2 を作成した。フォーム識別子 AMCPR1A。

## 設計判断

- モード: RWD・観察（共通ルールのみ・介入ヒューリスティック抑制）
- 先進医療を種別で PR（手技・処置・検査）/ EC（薬物・生物製剤・再生医療等製品）に2分割
- 実施回数: Interventions の `--DOSTOT` に転用、単位 `--DOSU=APPLICATION`（`TIMES` は UNIT CT C71620 非収載と一次照合で確認）
- 入院期間: `HO/HODUR`（整数→ISO8601 P{n}D）
- 転帰: `DS/DSDECOD`（NCOMPLT C66727 照合。治癒=RECOVERY・死亡=DEATH 標準、継続=ONGOING・中止=DISCONTINUED は sponsor 拡張）
- 評価結果: `RS/RSORRES`（sponsor: EFFECTIVE / NOT EFFECTIVE / UNKNOWN、RSTESTCD=OVRLEFF）
- 費用①〜④: `FA` に格納（FATESTCD 横展開、通貨 `FAORRESU=JPY` は sponsor 拡張・UNIT CT 非収載）
- 保険医療機関: `DM.SITEID`（submission value = 保険医療機関コード）
- 先進医療名: `--SPID`（全ドメイン provenance）兼 PR/EC `--TRT`、フォーム入力
- 患者番号: `DM.SUBJID`（診察券番号から自動代入・hidden）
- 年齢: `DM.AGE`(+AGEU=YEARS C66781)、性別: `DM.SEX`（男=M/女=F C66731）
- 総合計は自動算出のため非収集。1患者1先進医療のため RELREC 不要
- CT は Box cdisc_hapi_docker から取得し submission value を一次照合（捏造禁止）
